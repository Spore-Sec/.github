# Board Automation Scripts

These scripts manage the SporeSec Project V2 Board.

## Contents
- **sync_priority.ps1**: Syncs Priority labels (P0-P3) to the Board Field.
- **sync_status_by_labels.ps1**: Syncs Status based on labels (blocked -> Stuck).
- **project_map_final.json**: The Project Schema dictionary.
- **fetch_schema.graphql**: Query to update the schema map.

## Usage
Run via PowerShell or GitHub Actions.
